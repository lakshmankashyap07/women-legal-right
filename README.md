🌸 NLP Based Chatbot for Women’s Legal Rights
📌 Project Description
This project is a web-based chatbot designed to provide legal information and awareness to women. It helps users understand their rights related to domestic violence, harassment, dowry, workplace safety, and legal procedures in a simple and accessible way.
The chatbot uses Natural Language Processing (NLP) to understand user queries and provide relevant responses instantly. It acts as a digital legal assistant available 24×7.
🎯 Objectives
To spread awareness about women’s legal rights
To provide instant legal guidance
To create a user-friendly chatbot interface
To reduce dependency on legal experts for basic queries
To empower women using technology
🚀 Features
💬 Chat-based legal assistance
🧠 NLP-based query understanding
⚡ Instant responses using CSV dataset
🔐 User authentication (Login/Signup)
📩 Forgot password via email (SMTP)
🌐 Accessible on mobile & desktop
🗣️ Multilingual support (English & Hindi)

⚙️ How It Works
User enters a query in the chatbot
The system processes the input using NLP
Keywords are extracted and matched with dataset
Relevant legal response is generated
Response is displayed to the user
