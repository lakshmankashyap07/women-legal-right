<h1>🌸 NLP Based Chatbot for Women’s Legal Rights</h1>

<div class="section">
    <h2>📌 Project Description</h2>
    <p>
        This project is a web-based chatbot designed to provide legal information and awareness to women.
        It helps users understand their rights related to domestic violence, harassment, dowry, workplace safety,
        and legal procedures in a simple and accessible way.
    </p>
    <p>
        The chatbot uses Natural Language Processing (NLP) to understand user queries and provide relevant responses instantly.
        It acts as a digital legal assistant available 24×7.
    </p>
</div>

<div class="section">
    <h2>🎯 Objectives</h2>
    <ul>
        <li>Spread awareness about women’s legal rights</li>
        <li>Provide instant legal guidance</li>
        <li>Create a user-friendly chatbot interface</li>
        <li>Reduce dependency on legal experts for basic queries</li>
        <li>Empower women using technology</li>
    </ul>
</div>

<div class="section">
    <h2>🚀 Features</h2>
    <ul>
        <li>💬 Chat-based legal assistance</li>
        <li>🧠 NLP-based query understanding</li>
        <li>⚡ Instant responses using CSV dataset</li>
        <li>🔐 User authentication (Login/Signup)</li>
        <li>📩 Forgot password via email (SMTP)</li>
        <li>🌐 Mobile & desktop accessible</li>
        <li>🗣️ Multilingual support (English & Hindi)</li>
    </ul>
</div>

<div class="section">
    <h2>⚙️ How It Works</h2>
    <ol>
        <li>User enters a query</li>
        <li>System processes input using NLP</li>
        <li>Keywords are extracted</li>
        <li>Matched with dataset</li>
        <li>Relevant response is generated</li>
        <li>Response is displayed</li>
    </ol>
</div>
